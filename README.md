# RentFin
Esta es una APP diseñada para diseñar y optimizar tu renta financiera.  Sostenibilidad: Determina la fecha exacta de agotamiento del capital.  Poder Real: Descuenta la inflación para mostrar tu capacidad de compra futura.  Análisis de Estrés: Proyecta cómo afectan las oscilaciones del mercado a tu patrimonio.
